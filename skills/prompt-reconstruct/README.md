# prompt-reconstruct skill

This skill reconstructs raw user prompts into deterministic execution specs.

## Install target

- `~/.claude/skills/prompt-reconstruct/SKILL.md`

## Invoke

```text
/prompt-reconstruct <raw user prompt>
```

## Output contract

1. Reconstructed system prompt
2. Requirements map
3. Success criteria
4. Deliverables map
5. Validation checklist
6. Risks and assumptions
